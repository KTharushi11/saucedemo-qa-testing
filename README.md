# SauceDemo QA Testing Project

This project demonstrates **manual testing** of the SauceDemo web application, covering core functionalities such as login, cart operations, and checkout process.

---

### Application Under Test
 https://www.saucedemo.com

---

### Objective
To validate the functionality, usability, and reliability of the SauceDemo application through structured manual testing.

---

### Modules Tested

-  Login Functionality  
-  Cart (Add / Remove Items)  
-  Checkout Process  

---

### Testing Type

- Manual Testing  

---

### Deliverables

- Test Cases Documentation  
- Test Summary Report  
- Bug Report (with evidence/screenshots)

- UI Testing  
- Basic Security Testing
